# Spark4444.github.io

Collection of all my projects for github pages

# List of Projects

- CreateATable - A web app that allows you to create a table with custom data and download it as a PNG image. Or save it as a JSON file for later use.

- ChudjakAndDefenceOfAgartha - A simple ping-pong game about chudjak.

- MettatonNeoFangame - A browser fangame about Mettaton Neo from Undertale.

- Tab2Piano - A web app that converts a guitar tabs into piano visualization.

# Usage

Open the `https://spark4444.github.io/foldername/` link in your browser and replace `foldername` with the name of the folder you want to use.
